## Positionnement

> *« Je transforme tes pipelines bioinfo R&D en infrastructure de production fiable, auditable et augmentée par IA, avec les dashboards pour les piloter au quotidien. »*

**Tu fais** : industrialisation bioinfo end-to-end avec stack Nextflow + IA + dashboard.
**Tu ne fais PAS** : développement de nouveaux algorithmes ML/IA fondamentaux, biostats avancées, design d'expériences, IT générique non-bioinfo.

## Les 3 piliers

### Pilier 1 — Pipeline Nextflow industriel
Audit, refonte, optimisation, mise en production de pipelines Nextflow DSL2. Architecture modulaire, gestion erreurs, retry, monitoring, configs multi-environnements (AWS/Scaleway/Azure/local). Conformité ISO 15189 si labo clinique.

**Compétences mobilisées** : C1 (bioinfo) + C3 (pipeline industriel)

### Pilier 2 — IA / Skills Claude Code custom
Création de skills Claude Code spécialisés bioinfo (QC reports, audit trail, debug Nextflow, veille scientifique, batch effect detection, etc.). Agents pour automatiser les process labo. Intégration LLM dans les pipelines (scoring, annotation, summarization).

**Compétences mobilisées** : C1 (bioinfo) + C2 (prompt engineering)

### Pilier 3 — Dashboard / UI / monitoring
Construction de dashboards (Dash/Plotly, Next.js, Streamlit) pour visualiser les pipelines en production, traçabilité, audit trail, rapports automatisés, snapshots versionnés. Interfaces pour biologistes / chefs de labo / qualiticiens.

**Compétences mobilisées** : C5 (fullstack) + C3 (data pipeline)

## Risques identifiés

### Risque 1 — Le "couteau suisse" trap
Vouloir tout faire pour tout le monde = ne convaincre personne. **Le pitch doit être clair sur ce que tu fais ET ce que tu ne fais pas.**

### Risque 2 — Forfait mission vs jour facturé
Le forfait mission (15-60k€) est mieux pour Boris mais plus difficile à vendre. Commencer par Module A (5-8k€) comme produit d'appel court terme.

### Risque 3 — Scope creep dans les missions
Sur un forfait 50k€ pour 6 sem, le client va demander 30% de scope en plus gratuit. **Chiffrer les avenants dès la signature du contrat.**

### Risque 4 — Burnout AIMA + side
Cadre strict 50-55h/sem total. Stop sport+sommeil non négociables. Cf. [[feedback_coaching_style]] pattern burnout à surveiller.
